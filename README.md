# RentalSystem
Advanced Programming - Assignment 2



这个学期必修课，高级编程（Java），这是我的第二个作业
Topic：墨尔本租房小系统


界面UI：JavaFx
数据库：hsqldb
语言：Java


主要实现功能：

1. 数据导入导出功能（.txt）
2. 主页显示每一套房子的基本信息
3. 筛选功能：按照租房状态，租房类型等进行筛选
4. 在subpage，显示对应房子的具体信息，例如租房记录，房子简介
5. 在subpage，可以租房，退房，维护
6. 在subpage，可以创建新的房源，然后返回mainpage后，可以实时更新
7. 所有信息都会存在hsqldb
